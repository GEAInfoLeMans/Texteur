Démarrage du logiciel Word 2016

Méthode :

Pour démarrer le logiciel Word vous pouvez le faire :

  • à partir du menu [démarrer[tous les programmes[Microsoft Windows]]] ;

  • en cliquant sur le raccourci Microsoft Windows sur le bureau (si vous le voyez!) ;

  • en double cliquant sur un document Microsoft Windows (si vous en avez un dans vos dossiers).
