Style

Définition : Attribut

Un attribut est une caractéristique appliquée à un élément du texte.

Exemple : Attribut

Voici une liste de quelques attributs :

  • police et corps ;

  • gras, souligné, italique...

  • espace avant/après paragraphe ;

  • retrait de paragraphe ;

  • Interligne ;

  • espacement des caractères.

Définition : Style

Un style est ensemble d'attributs appliqués à un caractère, un paragraphe ou une section.

Style par défaut

Lorsque vous entrez du texte dans Word le style "normal" est appliqué par défaut.

Méthode : Changer le style d'un paragraphe

Les boutons du groupe Style du ruban accueil permettent de changer le style. Si vous amenez la souris sur l'un des styles, le paragraphe où se situe le curseur passera dans le style. Cette modification est validée si vous cliquez sur le style

Attention :

Le curseur doit être dans le paragraphe pour que la modification soit visible. Certains styles ne s'appliquent qu'à des mots ou bien à des mots sélectionnés. ils n'ont aucun effet sur le paragraphe.
