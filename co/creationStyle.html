Création d'un style, application du style à un autre paragraphe.

Méthode : Appliquer un style

Méthode : Création d'un style rapide

La création d'un style rapide se fait à partir du menu contextuel. Il faut alors donner un nom à ce style.

Méthode : Application du style à un autre paragraphe

Il faut mettre le curseur dans le paragraphe et cliquer dans le style correspondant (rapide).

Complément : Modifier un style
